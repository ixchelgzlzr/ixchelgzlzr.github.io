As part of the California Conservation Genomics Project, we are studying the genetic diversity of *Calasterella californica*. The first step to achieve this goal was to *de novo* assemble a reference genome. 

The [**assembly is available in NCBI (AstCali1)**](https://www.ncbi.nlm.nih.gov/datasets/genome/GCA_036924285.1/) under the name *Asterella californica*, which is the name of this species before been transferred to the new genus *Calasterella*.

Here, we make available the annotation of this reference genome based on transcriptomic and homology evidence.

### Downloads

[**Download annotation files here**]({{ 'assets/downloads/annotation_files.zip' | relative_url }}). This zipped directory includes:

- CDS fasta file.
- Genes fasta file.
- Gff3 annotation file.
- Protein faa file.
